# Interactive Resume - Praddumn Kumar

An elegant, interactive resume built with Next.js 15, featuring a beautiful notebook/flip-page design with smooth animations.

- **Flip-Page Animation**: Smooth page-turning effects like a real notebook
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Tech Stack**: Built with Next.js 15, React 19, TypeScript, and Tailwind CSS
- **Framer Motion**: Fluid animations and transitions throughout
- **Professional Design**: Vintage notebook aesthetic with gold accents
- **7 Interactive Pages**:
  1. Cover Page
  2. Personal Information
  3. Work Experience (updated with latest positions)
  4. Featured Projects
  5. Technical Skills
  6. Education & Achievements
  7. Contact & Social Links
